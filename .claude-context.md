## Project Status

**ACTIVE DEVELOPMENT** - Frontend-only architecture with remote Medusa backend.

**Business Model:** Soul Miner's Eden is a 65-acre regenerative farm in Georgia producing compost, livestock products, and agritourism experiences. This website serves as the digital storefront and booking engine. See [FARM-MASTER-PLAN.md](docs/FARM-MASTER-PLAN.md) for complete business strategy.

### What Exists
- Astro frontend with Tailwind CSS v4
- Remote Medusa backend on Railway
- Full product catalog (13 products)
- Cart and checkout flow (Medusa v2)
- **Land Management section** (`/land-management/*` - 4 pages)
- Cloudinary integration for images
- MUX video backgrounds (home hero, compost page)
- Admin tools (dashboard, brand guide, AI detection)
- Complete brand identity system
- Blog and recipes content system

### What's Next
- Remove "Certified" language (compliance)
- Build Voice/SEO admin pages
- DataForSEO integration
- See [TODO-NEXT-SESSION.md](TODO-NEXT-SESSION.md)
